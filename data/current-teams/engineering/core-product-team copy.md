# Core Product Team

The main product development team responsible for building and maintaining the core RouteOptix monolithic application running on Linux/RedHat. Develops the route optimization engine, delivery planning algorithms, and backend services.

## Responsibilities
- Route optimization algorithms and engine
- Delivery planning and scheduling logic
- Core business logic implementation
- Monolith application maintenance and evolution
- Windows desktop client for dispatchers (Java/Eclipse UI)
- Backend API services for route calculation
- Integration with Oracle database via ORM
- Real-time route updates and driver assignment

## Technologies
- **Backend**: C++, Python
- **Client**: Java, Eclipse RCP/UI framework
- **Database**: Oracle with ORM handler
- **Infrastructure**: Linux/RedHat, Docker
- **CI/CD**: Jenkins

## Team Structure
- 3 Backend Engineers (C++/Python)
- 2 Client Developers (Java/Eclipse)
- 1 QA Engineer
