# Database Platform Team

Small platform team providing database services, schema management, and data infrastructure for all development teams.

## Responsibilities
- Oracle database administration and maintenance
- Database schema design and optimization
- ORM configuration and support
- Database migration management via Flyway
- Query performance tuning and optimization
- Database backup and recovery
- Data security and access control
- Capacity planning and monitoring
- Database upgrade and patching

## Technologies
- **Database**: Oracle 19c
- **ORM**: Hibernate, custom ORM handlers
- **Migration**: Flyway
- **Scripting**: SQL, PL/SQL, Python
- **Monitoring**: Oracle Enterprise Manager, Grafana
- **Version Control**: Git

## Team Structure
- 1 Database Administrator (DBA)
- 1 Flyway/Migration Specialist
- 1 Database Engineer (ORM/Performance)

## Services Provided
- Database provisioning and configuration
- Schema migration services
- ORM framework support and training
- Performance optimization consultancy
- Database access and security management
