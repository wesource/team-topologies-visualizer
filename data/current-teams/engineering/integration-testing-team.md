# Integration Testing Team

Verifies the entire system integration, testing components provided by all development teams within the company.

## Responsibilities
- End-to-end integration testing
- System integration verification
- Automated test suite development and maintenance
- Performance and load testing
- Regression testing
- Test environment management
- Test data management
- Defect tracking and reporting

## Technologies
- **Functional Testing**: UFT/QTP, Selenium, Postman
- **Performance Testing**: LoadRunner, K6
- **Test Management**: TestRail, JIRA
- **Automation**: Python, Java
- **CI/CD**: Jenkins
- **Environment**: Docker, Linux

## Team Structure
- 3 QA Engineers (manual and automated testing)
- 1 Test Automation Engineer
- 1 Performance Test Engineer
