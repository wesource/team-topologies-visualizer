# Web Product Team

A small focused team developing and maintaining the RouteOptix dispatcher and planner web interface used by logistics coordinators and dispatchers to plan routes and manage deliveries.

## Responsibilities
- Dispatcher web interface development
- Route planning and visualization UI
- Interactive map features and route display
- Delivery tracking dashboard
- User experience implementation for logistics coordinators
- Responsive design for tablet/desktop use
- Frontend-backend integration via REST APIs
- UI/UX design and prototyping

## Technologies
- **Frontend**: Angular, JavaScript, HTML5, CSS3
- **Backend**: Java (lightweight services)
- **Build Tools**: npm, Webpack
- **Deployment**: Docker

## Team Structure
- 2 Frontend Engineers (Angular/JavaScript)
- 1 UX/UI Designer
